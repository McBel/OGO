Справочник ЛС

Список уникальных идентификаторов присваиваемых неким объективным сущностям, в разрезе которых, производится учет:


  • оказания услуг;
  • производства работ;
  • заключенных договоров;
  • выставления счетов;
  • проведения взаиморасчетов;
  • и т.п.

Каждая запись(ДВИЖЕНИЕ) отражаемая в учете должна быть обязательно связана с каким либо Лицевым счетом и возможно(но может быть не обязательно) с Договором.

Основное представление в виде кода
Код уникальный 21 симв алфавит ===только буквы и цифр

Номер только из цифр
строковый тип (внутри)
допустим формат "Префикс"_" номер"_"суффикс"
Фильтрация незначащих символов (убираем пробелы и лидирующие ноли в номере)
В префиксе обязательно наличие буквы иначе это не префикс )
Префиксы и суффиксы только в виде исключений (по умолчанию только номер и только из чисел)
